# Skvare React Block Module

The Skvare React Block module allows you to embed a React application within a Drupal block. This provides you with the flexibility to create interactive and dynamic user interfaces.

## Requirements
- Node.js version 20 or higher
- NPM (Node Package Manager)

## Building the React App
To build the React app, use the provided npm scripts:

`npm install`

`npm run build`: Compiles the React app in production mode.

`npm run build:dev`: Compiles the React app in development mode.


These scripts will run Webpack to bundle your React code and generate the necessary JavaScript and CSS files in the js/dist directory.
